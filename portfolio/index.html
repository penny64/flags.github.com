GOAPy

GitHub - Example

GOAPy is a pure-Python implementation of Goal-Oriented Action Planning, a generic architecture for autonomous AI.

By taking the current state of the AI (e.g.: Hungry? Tired?) and their desired state (e.g.: Not tired) a "plan" is generated to take the AI from their current state to their goal state by considering a list of actions. In this case, the AI will discover that in order to sleep, they must first find and consume food, potentially leading to a complex course of action that involves going shopping, cooking, ordering pizza, taking a shower, etc.


Reactor 3 - The Dead World

GitHub

Open-world survival simulator set in a post-apocalyptic Europe. Features complex NPC behavior, detailed damage model, and procedurally generated assets. Developed for two years using Python in addition to Cython-compiled modules.

  • Living, breathing world: NPCs loot, manage items, and participate in a large-scale battle between factions.
  • Day/night cycle with weather effects.
  • Particle system capable of dynamic lighting.
  • Huge open-world filled with towns, military outposts, and other landmarks.
  • Basic mod support - create items and lifeforms to inhabit the world and script their behaviors.
