logtime.py

I created logtime.py after reading about "time management" software on Facebook. I was interested enough to look up possible open-source alternatives, but felt they were too intrusive.

logtime does not monitor your system or spy on you in any way.

Requirements

  • Python 2.7+ (Might work with earlier versions.)
  • CairoPlot*
  • cyclelogs.py (Optional)

*CairoPlot's website is garbage. This might be better.

Usage/Output

logtime.py [option] [file]
  • advance
  • build
  • close
  • compile
  • list

advance 

logtime.py advance

Starts a new log, but carries over the last entry from the previous day. Requires cyclelogs.py.

build 

logtime.py build

Creates the logfile. Erases all content.

close

logtime.py close

Closes the logfile for archiving.

compile 

logtime.py compile [output]

Outputs a piechart.

list 

# logtime.py list
00:00:01 Tue. February 2012: Offline

Lists all logged entries made in the current day.
