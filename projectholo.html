Project Holo

Project Holo is a set of scripts I've written to work in sync with cron. These are standard tasks that I just got tired of performing manually.

Warning

Some people might get upset over my "updates" script, thus, I won't be posting the second half of it, which runs hourly via root's cron (pacman -Sy). Use at your own risk!
